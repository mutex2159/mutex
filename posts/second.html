소리를 이용한 거리 측정: 초음파 센서와 아두이노의 만남

초음파 센서는 소리의 반사를 이용해 물체와의 거리를 측정하는 장치입니다. 일반적으로 사용되는 초음파 센서 중 하나는 HC-SR04 모델로, 송신부에서 고주파의 음파(초음파)를 발사한 뒤, 물체에 반사된 음파를 수신부에서 받아들여 거리를 측정합니다.

초음파 센서는 트리거(Trigger) 신호를 받으면 음파를 발사하고, 에코(Echo) 핀에서 반사된 신호를 수신합니다. 이 시간차를 계산해 물체와의 거리를 구할 수 있습니다.

초음파는 공기 중에서 일정한 속도로 이동합니다. 이 속도는 약 343 m/s(20°C 기준)로, 초음파가 물체에 닿아 반사되어 다시 센서로 돌아오는 데 걸리는 시간을 측정하면 물체와의 거리를 계산할 수 있습니다. 기본적으로 거리 계산은 다음 수식을 통해 이루어집니다:

$$$ d = \frac{t \times v}{2} $$$

$$ d $$: 물체와 센서 사이의 거리 (m)

$$ t $$: 음파가 송신되고 다시 수신될 때까지의 시간 (s)

$$ v $$: 음파의 속도 (343 m/s, 온도에 따라 달라짐)

여기서 나누기 2를 하는 이유는 초음파가 물체에 닿았다가 되돌아오는 왕복 거리를 측정하기 때문입니다. 즉, 실제 거리 $$ d $$는 왕복 시간에 기반해 계산됩니다.

아두이노를 사용해 초음파 센서로 거리를 측정하는 간단한 방법을 살펴보겠습니다. HC-SR04 센서를 아두이노에 연결하고, 트리거 핀을 통해 음파를 송신하고 에코 핀에서 수신된 시간을 바탕으로 거리를 계산합니다.

const int trigPin = 9;
const int echoPin = 10;
long duration;
float distance;

void setup() {
  Serial.begin(9600);
  pinMode(trigPin, OUTPUT);
  pinMode(echoPin, INPUT);
}

void loop() {
  // 트리거 핀에 10μs의 신호를 보내 음파를 발사
  digitalWrite(trigPin, LOW);
  delayMicroseconds(2);
  digitalWrite(trigPin, HIGH);
  delayMicroseconds(10);
  digitalWrite(trigPin, LOW);
  
  // 에코 핀에서 반환된 시간을 측정
  duration = pulseIn(echoPin, HIGH);
  
  // 시간에 기반하여 거리 계산
  distance = (duration * 0.0343) / 2;

  // 측정된 거리 출력
  Serial.print("Distance: ");
  Serial.print(distance);
  Serial.println(" cm");
  
  delay(1000);  // 1초 대기
}

이 코드에서는 트리거 핀에 10마이크로초의 신호를 보내 음파를 발사한 후, 에코 핀에서 반환된 시간을 바탕으로 거리를 계산합니다. pulseIn() 함수는 에코 핀에서 고신호를 받는 데 걸린 시간을 마이크로초 단위로 반환합니다. 이 시간을 기반으로 거리 계산에 사용되는 단위는 cm입니다.
